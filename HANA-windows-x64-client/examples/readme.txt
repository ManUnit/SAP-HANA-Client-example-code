SAP HANA Database Client Examples
=================================

The subfolders contain examples of applications using selected SAP HANA
database clients.

For further instructions on the examples in the sub-folders, consult the
SAP HANA Client Interface Programming Reference for a selected client.

  .NET Core:        https://help.sap.com/viewer/f1b440ded6144a54ada97ff95dac7adf/latest/en-US/837edd3925884d9cb0dc3ff8cafac77e.html
  Go (golang):      https://help.sap.com/viewer/f1b440ded6144a54ada97ff95dac7adf/latest/en-US/2877e9c950a247369afe27a5cd68fe99.html

For further instructions showing how to create applications that make use
of selected SAP HANA database clients, see the tutorial:
"Use Clients to Query an SAP HANA Database" at:

  https://developers.sap.com/mission.hana-cloud-clients.html
